# Cavekitty16.github.io

This repository houses the source files for [https://cavekitty16.github.io](https://cavekitty16.github.io), a personal website showcasing projects, resume, and other web development resources.

## Contents

- **index.html** – Home page for the site.
- **about**, **projects**, **resume**, and **webDev** pages for various sections of the site.
- **css/** – Stylesheets used by the pages.
- **scripts/** – JavaScript powering interactive features.
- **data/** – JSON data used by the site.
- **images/** – Images displayed on the site.
